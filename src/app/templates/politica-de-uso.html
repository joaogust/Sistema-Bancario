Política de Uso e Privacidade

Última atualização: 25 de outubro de 2025

1. Coleta de Dados

Para fins de simulação de uma conta de usuário, esta plataforma acadêmica coleta os seguintes dados (fictícios):

  • Nome de usuário (fictício).
  • Endereço de e-mail (necessário para login e recuperação de senha).
  • Senha (armazenada de forma segura usando hashing).

Reiteramos: não insira dados pessoais reais, com exceção de um e-mail válido, caso deseje testar a funcionalidade de recuperação de senha.

2. Uso dos Dados Coletados

Os dados coletados são utilizados exclusivamente para os seguintes fins:

  • Permitir o login e logout na plataforma (autenticação).
  • Simular a personalização da "Área do Cliente" (ex: "Olá, [Nome]").
  • Simular transações entre contas fictícias dentro do sistema.

Seus dados não serão compartilhados, vendidos, alugados ou utilizados para qualquer fim comercial, de marketing ou de análise fora do escopo deste projeto acadêmico.

3. Armazenamento e Segurança

Aplicamos práticas de segurança para proteger os dados da simulação, conforme descrito em nossa Página de Segurança. Isso inclui o uso de criptografia HTTPS e hashing de senhas.

4. Cookies

Podemos utilizar cookies de sessão essenciais para manter o usuário logado durante a navegação. Estes cookies são temporários e são excluídos quando você fecha o navegador. Não utilizamos cookies para rastreamento ou publicidade.
